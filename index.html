Overview

The AURA (Automated Utility & Resource Assistant) is an all-in-one productivity tool designed to streamline microsite operations through automation and comprehensive project management capabilities. With a suite of integrated modules, AURA enables seamless data management, real-time monitoring, efficient project updates, and intuitive AUX tracking. Equipped with secure authentication, and advanced data handling features, AURA empowers Microsite productivity, and maintain precise control over microsite hours utilization.

📋

Project Details Card

A replacement to existing project details quip which facilitates daily project updates and retrievals through a streamlined interface, ensuring project documentation stays visible for the team.

📊

NPT Tracker

Enables seamless NPT management with robust import/export capabilities across CSV and AWS platforms, providing complete control over AUX data handling.

🗄️

Database Manager

Features search functionality with AWS integration for efficient project time tracking and comprehensive NPT data retrieval.

📈

Flash Data Dashboard

Delivers real-time microsite metrics through an intuitive visual interface, enabling instant data monitoring.

⚡

Efficiency Tracker

Provides real-time AUX monitoring through an interactive dashboard, helping optimize resource utilization and effective reporting.
